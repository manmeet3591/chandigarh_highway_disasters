# Chandigarh Highway Disasters POC

Interactive dashboard for mapping high-fatality blackspots and highway infrastructure in Chandigarh, India.

## Live Dashboard
View the interactive POC at: [manmeet3591.github.io/chandigarh_highway_disasters](https://manmeet3591.github.io/chandigarh_highway_disasters)

## Project Components
- **Interactive Map:** Built with Leaflet.js using GeoJSON layers.
- **Data Layers:**
  - `chandigarh_highways.geojson`: Full highway network from OpenStreetMap.
  - `chandigarh_blackspots.geojson`: High-fatality zones identified by police reports.
- **Tech Stack:** HTML5, CSS3, JavaScript, Leaflet.js.

## Deployment
This project is set up for GitHub Pages. 
1. Push the contents of the `web/` directory to the `main` branch.
2. Enable GitHub Pages in the repository settings, pointing to the `web/` folder or the root if the files are moved there.
