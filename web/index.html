Project Overview

This Proof of Concept (POC) visualizes critical hazard zones across the Chandigarh metropolitan region. By integrating OpenStreetMap highway geometries with official traffic safety data, we identify the specific nodes where infrastructure design and high traffic volume converge into "disaster zones."

Methodology

  • Spatial Layer: Extracted using Overpass API (OSM) for the Chandigarh Bounding Box.
  • Disaster Layer: Compiled from Chandigarh Police Road Accident Analysis Wing (RSAW) data.
  • Target Hazards: Fog-related pile-ups, Monsoon cave-ins, and high-speed collision points.
